This directory contains icons for the Integrated Terminal Desktop application.

Required icons:
- app.ico - Main application icon
- terminal.ico - Terminal icon
- explorer.ico - File explorer icon
- taskbar.ico - Taskbar icon
- widget.ico - Widget icon
- search.ico - Search icon
- settings.ico - Settings icon

Icons should be provided in multiple resolutions:
- 16x16
- 32x32
- 48x48
- 64x64
- 128x128
